Projects


Reddit Link Bot

Using PRAW API and Python, created a Reddit Bot that identified comments with Reddit links, parsed the contents of the link, and posted a CSS-formatted reply. The purpose was to help users on mobile or slow internet by saving them unnecessary clicks. Although the bot is no longer active or hosted, the reception was relatively positive on from the Reddit community.


Machine Learning for League of Legends

Following in the footsteps of Deep Blue, I am working on a project for training a machine for League of Legends. The first step is focused on predicting outcomes given a game state. By being able to predict outcomes based on game states, a machine could determine the best objective to pursue to maximize win rate.
